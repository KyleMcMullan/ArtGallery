# ArtGallery
Prototype simulation of an art gallery containing 3 of Xenoyr Campbell's artworks.

To run, click on 'Clone or Download'
Click 'download zip'
Once the zip file is downloaded, open it and extract it anywhere you can find it
Open the unzipped folder, open ArtGallery-master, open Art Gallery
Double click on 'Art Gallery - Xen.exe'

wasd and arrow keys to move
Mouse to look around
There's no exit button so alt+tab to leave the screen.
